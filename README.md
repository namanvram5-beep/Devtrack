How to run the project

Go to the project folder
Run this command:
   python manage.py runserver
Open browser or Postman and use:
   http://127.0.0.1:8000/

API endpoints

Reporter

POST /api/reporters/
GET /api/reporters/
GET /api/reporters/?id=1

Issue 

POST /api/issues/
GET /api/issues/
GET /api/issues/?id=1
GET /api/issues/?status=open
